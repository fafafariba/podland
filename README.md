# Podland

[Podland](https://podland.herokuapp.com) is a podcast library web application inspired by [Stitcher](https://www.stitcher.com). It provides a virtual space for podcast listeners to track their favorite shows, craft their own playlists, and discover new shows. 

![alt text](https://github.com/fafafariba/podland/blob/master/docs/podland.png?raw=true "Podland")


## Features

* Diverse collection of podcasts with ability to stream episodes continuously

* 'Featured' and 'Popular' suggestions

* Subscriptions to track entire podcast series

* Custom playlist creation

* Personalized accounts with secure authentication

* Contemporary aesthetic appeal
