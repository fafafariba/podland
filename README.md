# Podland

[Podland](https://podland.herokuapp.com) is a podcast library web application inspired by [Stitcher](https://www.stitcher.com). It provides a virtual space for podcast listeners to track their favorite shows, craft their own playlists, and discover new shows. 

![alt text](https://github.com/fafafariba/podland/blob/master/docs/podland.png?raw=true "Podland")


## Features

* Diverse collection of podcasts with ability to stream episodes continuously

* 'Featured' and 'Popular' suggestions

* Subscriptions to track entire podcast series

* Custom playlist creation

* Personalized accounts with secure authentication

* Contemporary aesthetic appeal

## Future Implementation

Though Podland presents several features already, it's open to growth and improvement. Some upcoming expansions include the following.

### Search

Searching provides a convenient way for users to quickly navigate to specific podcasts. An advanced search feature will offer the ability to search by categories.

### Feed 

Users will appreciate a live feed of the the most recently published podcast episdoes, provides yet another avenue for users to explore new shows. One potential way to accomplish this would be to utliize [Audiosear.ch](https://www.audiosear.ch)'s API for podcasts and radio.
