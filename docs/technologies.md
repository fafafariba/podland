# Technologies

Podlands 

## Back-End

### Rails

Podland runs on the MVC framewwork Ruby on Rails.

### PostgreSQL

All of Podland's data was stored in PostgresQL, an open source database.

### BCrypt

Secure authentication was setup with the use of BCrypt for password-salting and hashing.

## Front-End

### React.js

The front end framework for Podland is React.js in order to faciliate the creation of a single page app. 

### Redux

Flux architecture, Redux, provided the structure of the process.

### jQuery

jQuery provided the means to make Ajax request to the database without sending HTTP requests.

### SCSS

More 'Sassy' CSS for styling.

### Yarn

[Yarn](https://yarnpkg.com/en/), a recent dependency manager was used in lieu of the conventional NPM.

### Babel

Babel compiler was used to transpile ES6 to ES5 for browsers.

### Libraries

[React-Router](https://github.com/ReactTraining/react-router) for seamless transitioning from page to page on a single page app

[React-Modal](https://github.com/reactjs/react-modal) for modals

[React-Player](https://www.npmjs.com/package/react-player) for the continuous playing audio player
